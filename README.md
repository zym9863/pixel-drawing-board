# 像素绘画板 (Pixel Drawing Board)

[中文](README.md) | [English](README_EN.md)

一个简洁、美观的像素风格绘画工具，让您轻松创建像素艺术作品。

## 功能特点

- **直观的绘画界面**：16x16像素网格，简单易用
- **多种绘画工具**：
  - 铅笔工具：自由绘制像素
  - 橡皮擦：擦除已绘制的像素
  - 填充工具：填充相连的相同颜色区域
  - 一键清除：重置整个画布
- **丰富的颜色选择**：
  - 内置10种预设颜色
  - 自定义颜色选择器
  - 当前颜色显示
- **现代化界面**：
  - 赛博朋克风格设计
  - 霓虹灯效果
  - 响应式布局，支持移动设备
- **触摸屏支持**：在平板和手机上也能流畅使用

## 技术栈

- HTML5
- CSS3 (动画、渐变、变量等现代特性)
- 原生JavaScript (无依赖)
- Font Awesome 图标

## 使用方法

1. 选择一种颜色（使用预设颜色或自定义颜色选择器）
2. 选择绘画工具（铅笔、橡皮擦或填充工具）
3. 在网格上绘制您的像素艺术
4. 使用清除按钮可以重置画布

## 本地运行

只需将项目文件下载到本地，然后在浏览器中打开`index.html`文件即可使用。

```bash
# 克隆仓库（如果使用Git）
git clone https://github.com/zym9863/pixel-drawing-board.git

# 进入项目目录
cd pixel-drawing-board

# 使用浏览器打开index.html
```
