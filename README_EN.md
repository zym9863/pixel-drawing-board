# Pixel Drawing Board

[中文](README.md) | [English](README_EN.md)

A clean and beautiful pixel-style drawing tool that allows you to easily create pixel art.

## Features

- **Intuitive Drawing Interface**: 16x16 pixel grid, simple and easy to use
- **Multiple Drawing Tools**:
  - Pencil tool: freely draw pixels
  - Eraser: erase drawn pixels
  - Fill tool: fill connected areas of the same color
  - One-click clear: reset the entire canvas
- **Rich Color Selection**:
  - 10 built-in preset colors
  - Custom color picker
  - Current color display
- **Modern Interface**:
  - Cyberpunk style design
  - Neon light effects
  - Responsive layout, supports mobile devices
- **Touchscreen Support**: Works smoothly on tablets and phones

## Tech Stack

- HTML5
- CSS3 (animations, gradients, variables, and other modern features)
- Native JavaScript (no dependencies)
- Font Awesome icons

## How to Use

1. Select a color (using preset colors or custom color picker)
2. Choose a drawing tool (pencil, eraser, or fill tool)
3. Draw your pixel art on the grid
4. Use the clear button to reset the canvas

## Local Setup

Simply download the project files to your local machine, then open the `index.html` file in your browser to use.

```bash
# Clone the repository (if using Git)
git clone https://github.com/zym9863/pixel-drawing-board.git

# Enter the project directory
cd pixel-drawing-board

# Open index.html in your browser
```
